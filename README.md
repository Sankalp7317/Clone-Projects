# Clone Projects

A collection of website clones built using **HTML, CSS, and JavaScript** for learning, practice, and improving frontend development skills.

## Projects

### Netflix Clone

A responsive Netflix landing page clone built with HTML, CSS, and JavaScript.

**Features**

* Responsive design
* Modern UI inspired by Netflix
* Hero section and content layouts
* Clean and organized code structure

## Tech Stack

* HTML5
* CSS3
* JavaScript

## Purpose

This repository serves as a showcase of frontend cloning projects created to strengthen web development skills and recreate real-world user interfaces.
